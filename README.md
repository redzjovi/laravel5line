## Synopsis

Laravel 5.4 with Line Bot Messages.

## Installation

1. Edit .env and set DB_DATABASE, DB_USERNAME, DB_PASSWORD
2. In terminal, run migration to generate table<br />
"php artisan migrate:refresh --seed"<br />
